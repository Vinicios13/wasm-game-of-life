## How to run

### in the root folder
```
wasm-pack build
```

### in the www folder
```
npm install && npm start
```

## prerequisites
<a href="https://www.rust-lang.org/tools/install"> rustup, rustc, and cargo </a>

<a href="https://rustwasm.github.io/wasm-pack/installer/"> wasm-pack </a>
